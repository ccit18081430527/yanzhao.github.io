用户认证模块

登录功能

需求ID: AUTH-001
优先级: 高
描述: 用户通过用户名密码登录系统 

@startuml
actor 用户
participant "认证服务" as auth
用户 -> auth: 提交凭证
auth -> 数据库: 验证用户
数据库 --> auth: 用户数据
alt 验证成功
    auth --> 用户: 登录成功
else 验证失败
    auth --> 用户: 错误提示
end
@enduml
